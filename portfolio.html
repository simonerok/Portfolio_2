Project 13C.01.01

Choosing a font

Working with fonts a great tool to use is "TypeScale.com". The program allows you to test sizes, font families and pair fonts. Font hiarachy is an important part of a website. We are working with display fonts and paragraph text. Usually iis better to choose a simple dispaly font like a sans serif and not a crazy display font. Choosing fonts its better to go with two types of fonts and work with boldness and making the text itallic etc.

Mockups

For 13c.01.01

Here are some mokups showing how the fonts are used and how it looks on an actual website. I have worked with sans-serifs because they are very easy to read and contribute to a modern look for a website.

Fonts

I have chosen my primary font to be "Bebas Neue". I like it because it is very modern yet it is also good for displaying in a bigger scale. For the paragraph text i have worked with both "montserrat" and "Inter" but i chose to go with "Inter" because it gives a very clean look and montserrat look very compressed when under 18px so that some of the words melt together which is not good in a paragraph text.

Details

Here are some details involving the ui components and hover effects.

Default look

Buttons

I have worked with buttons, a simple look is going to work in most cases because you dont want to confuse the reader. On the contrary buttons are made to draw attention which is why it has to stand out in some way. Thats why colors are important. In my case i have chosen a light color for the button which makes it stand out on the darker background. You could also draw attention by choosing a crazy color like pink or orange.

Hover effect

Hover effect

In addition i have worked with hover effects for the buttons. This is a good way to show the reader that they can interact with the button. Ive chosen a simple hover effect that makes an arrow appear on the button when the mouse is hovered. You could also make a hover effect that changes the color of the button but i chose a simple hover effect and ofcourse to make the curser a "pointer".

Summary: What have i learned from this theme

To summarize, I have learned a lot about design, including a good understanding of the use of fonts and colors. I have learned to implement animations, better hover effects, and create a "dark mode". Additionally, I have discovered the importance of layout and the use of 'cards', which I had not properly utilized before.
