Web animations

I have used an animation that fades in the website using opacity that goes from 0 % to 100% which creates this "blink" effect. The animation is made with JavaScript by importing the animation and then use it and in CSS by using keyframes.

Codesnippet for dark theme

Dark mode

I have created a button that can switch between the dark mode and light mode. One important thing I have learned while creating the dark mode is that it doesn't necessarily have to be a transition from dark to light or from black to white.

Summary: What have i learned from this theme

To summarize, I have learned a lot about design, including a good understanding of the use of fonts and colors. I have learned to implement animations, better hover effects, and create a "dark mode". Additionally, I have discovered the importance of layout and the use of 'cards', which I had not properly utilized before.
